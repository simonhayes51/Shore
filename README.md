# Shoreline Pizza Co. Landing Site

This repository contains the full static landing page build for **Shoreline Pizza Co.**

## Included files

- `index.html` — page markup
- `styles.css` — responsive styles
- `script.js` — optional reduced-motion-aware hero parallax
- `assets/shoreline-logo-horizontal.svg` — horizontal logo
- `assets/shoreline-logo-round.svg` — round logo badge
- `server.js` — minimal static server
- `package.json` — `npm start` script

## Run locally

```bash
npm start
```

Server binds to `PORT` when provided, otherwise defaults to `3000`.
